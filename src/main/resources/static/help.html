How It Works

Bible Search uses AI-powered intent classification to automatically choose the best search method. The system understands your query using embeddings (semantic vectors) and routes it appropriately.

Search Modes
Keyword Search

For specific words, names, places. Triggered by patterns like ~가 나오는, ~라는 지명, verses containing

Examples: 가사라는 지명이 나오는 구절, 모세가 등장하는, verses with shepherd

Semantic Search

For concepts, themes, questions. Understands meaning and finds conceptually related verses.

Examples: 사랑에 대한 말씀, comfort in suffering, God's faithfulness

Hybrid Search

Short queries (1-2 words) automatically use both methods.

Examples: 가사, Moses, 믿음

Scoped Search

Add a book scope using ~에서 or in ~. Supports testaments, book groups, single books, or multiple books.

  • Testament: 신약에서, 구약에서, in the New Testament
  • Book Groups: 사복음서, 모세오경, 바울서신, 시가서, 대선지서
  • Single Book: 로마서에서, 창세기에서
  • Multiple Books: 이사야, 예레미야에서

Examples: 사복음서에서 사랑이 나온 구절, 로마서에서 복음의 정의, 바울서신에서 믿음에 관한

Tips

  • Adjust score threshold — Lower for more results, higher for precision
